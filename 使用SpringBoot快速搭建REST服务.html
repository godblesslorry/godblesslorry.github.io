使用SpringBoot快速搭建REST服务

Posted on 2017-04-20 | Comments: | Views:

Spring Boot是由 Pivotal 团队提供的全新框架，其设计目的是用来简化 Spring 应用的初始搭建以及开发过程。该框架使用了特定的方式来进行配置，从而使开发人员不再需要定义样板化的配置。通过这种方式，我们可以更加快速地搭建 web 应用或者是 REST 服务。

创建新项目 New Project->Spring Initializr ->web[web]
然后配置项目的 maven pom.xml文件

运行 SpirngbootdemoApplication 的 main 方法，就能启动项目。

构建项目的目录结构

├───java
│ └───com
│ ├───aspect
│ ├───controller
│ ├───dao
│ ├───entity
│ ├───enums
│ ├───exception
│ ├───handler
│ ├───service
│ └───utils
└───resources

resources 目录包含了项目的基本配置，包括数据库连接和 url 前缀配置
application.yml 是主配置文件，利用它可以将项目在开发环境和生产环境自由切换
