自动更新package.json至最新版本

Posted on 2017-12-11 | Comments: | Views:

当我们运行陈旧项目时，常常会因为各个依赖包版本过时而导致 npm 安装失败，手动修改各个依赖包的版本号当然是解决问题的方法之一，但是面对大量的参数逐一查找最新版并修改未免太繁琐。

npm update

使用 npm update 可以自动更新依赖包，但是这样更新后并不会同步修改 package.json 文件，下一个开发者使用这个项目时还是会面对版本过时的困扰。

npm-check-updates

npm-check-updates 工具可以很方便的将 package.json 中的依赖包版本号更新为最新版本。使用方法非常简单：

安装

npm install npm-check-updates -g

检查最新依赖版本

npm-check-updates 或 ncu

更新 package.json 中的依赖至最新版本

ncu -u 或者 ncu -a
